# Lacrei Saúde - Front-end Challenge

Aplicação desenvolvida para o desafio técnico de Front-end da Lacrei Saúde.

O projeto tem como objetivo criar uma experiência digital acessível, responsiva e acolhedora, facilitando a conexão entre pessoas e profissionais de saúde.

## 🚀 Tecnologias utilizadas

- Next.js
- React
- TypeScript
- Styled Components
- Jest
- Testing Library
- Lucide React

## 📌 Funcionalidades

- Página inicial com apresentação da plataforma
- Navegação responsiva para desktop e mobile
- Página de ajuda com informações para usuários
- Componentes reutilizáveis
- Menu de acessibilidade
- Layout adaptável para diferentes telas
- Footer com navegação
- Testes automatizados dos componentes principais

## ♿ Acessibilidade

Foram aplicadas práticas de acessibilidade como:

- Uso de elementos semânticos HTML
- Labels e atributos ARIA
- Navegação acessível
- Foco visível para teclado
- Estrutura preparada para leitores de tela

## 📱 Responsividade

A aplicação foi desenvolvida pensando em diferentes dispositivos:

- Desktop
- Tablet
- Mobile
